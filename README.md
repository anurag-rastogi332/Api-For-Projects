# Api-For-Projects

This repository contains multiple Node.js API projects, each in its own folder:

- contact api
- File Uploader
- User Contacts api
- video-stream api

Each project is ready for deployment and follows best practices for environment variables and open-source licensing.

## Getting Started

1. Clone the repository
2. Copy the `.env.example` file in each project to `.env` and fill in your secrets
3. Install dependencies with `npm install` in each project folder
4. Run the server with `node server.js` or `npm start`
